# AGENTS

本文件定义 Bio-Agent 系统中可用的 subagent 及其能力。

## 可用 Subagent 列表


### task_decomposition
- **职责**: 计划分解
- **能力**: 将复杂计划分解为子任务、识别依赖关系
- **适用场景**: 只有计划，而未细化为标准化的任务清单
- **输出**：标准化的任务清单

### general_qa
- **职责**: 通用问答
- **能力**: 回答通用问题、生物信息学知识问答
- **适用场景**: 简单的问答、知识查询
- **输出**：问题的答案

### immunity
- **职责**: 免疫学分析
- **能力**: TCR/BCR 分析、免疫组库分析
- **适用场景**: 免疫学相关的研究任务
- **输出**：问题分解结果、深度研究报告、科学假设、计划、评估报告


### result_evaluator
- **职责**: 结果评估
- **能力**: 评估任务执行结果、生成报告
- **适用场景**: 已经有了任务执行结果，需要对结果进行评估
